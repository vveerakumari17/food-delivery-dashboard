# 🍔 Food Delivery Sales Analysis Dashboard

## 📌 Project Overview

This project is an interactive Power BI dashboard designed to analyze food delivery business performance. It provides valuable insights into sales, orders, customer behavior, delivery performance, and revenue trends to help businesses make data-driven decisions.

---

## 🎯 Objectives

- Analyze overall sales performance
- Monitor total orders and revenue
- Identify top-performing restaurants
- Track customer ordering behavior
- Analyze delivery efficiency
- Discover sales trends over time

---

## 📊 Dashboard Features

- 📈 Total Sales KPI
- 🛒 Total Orders
- 💰 Total Revenue
- 🚚 Delivery Status Analysis
- 🍕 Category-wise Sales
- 🏪 Restaurant Performance
- 👥 Customer Analysis
- 📅 Monthly Sales Trend
- 🌍 Location-wise Orders
- 📊 Interactive Filters & Slicers

---

## 🛠 Tools Used

- Power BI Desktop

- DAX

- Excel/CSV Dataset
